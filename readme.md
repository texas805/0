## ğŸ“ Assignment Guidelines

1. **Naming Convention**
   - Use descriptive folder names: `assignment-01`, `assignment-02`, etc.
   - Use lowercase with hyphens for file names: `index.html`, `styles.css`, `script.js`

2. **Code Quality**
   - Write clean, readable, and well-commented code
   - Follow HTML5 and CSS3 best practices
   - Use proper indentation and formatting

3. **Submission**
   - Push all assignments to your repository
   - Include a README.md in each assignment folder explaining what the assignment does
   - Ensure all files are properly committed and pushed

4. **Deadlines**
   - Check with your instructor for specific deadlines
   - Late submissions may be penalized
